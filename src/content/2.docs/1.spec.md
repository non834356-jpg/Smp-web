This page introduces the service specifications of the jao Minecraft Server.

## About Login

### Server Address

It is listed at the bottom of pages on this website.  
It is also published on [minecraft.jp](https://minecraft.jp/servers/play.jaoafa.com) and [monocraft.net](https://monocraft.net/servers/4ovU0v9PkdyJbNJVngf7).

### Version

The server version is **1.18.2**.  
You can also join using the following versions:

- Minecraft Java Edition: 1.9.x – 1.18.2

However, available blocks and items follow version 1.18.2.  
So if you join using another version, some blocks or items may not work properly.

### Restricted Logins

- **Multiple Accounts**  
  Using alternate (sub) accounts is prohibited. This is to prevent abuse and reduce server load.  
  The first account you log in with will be treated as your main account.  
  After that, logging in with other accounts is not allowed.  
  If you must use a different account due to special circumstances, please contact us via the official Discord server `#support`.

- **Login from Outside Japan**  
  This server is operated as "Japanese Only".  
  If the system detects access from outside Japan, login will be blocked.

### If You Cannot Log In

The server may be down for some reason.  
Check the [status page](https://status.jaoafa.com) or the official Discord channels `#info` and `#announcement`.  
Also, check the [FAQ](/support/faq) page.

## Game Modes

The server starts in **Creative Mode** by default.  
You can switch modes using commands like `/gamemode` or `/g`.

Spectator Mode is available only for players with **Regular rank or higher**.

---

## Custom Features

This server has unique features using plugins, different from vanilla servers:

- Water and lava spreading is restricted in the central city (free build area).
- Nether portals cannot be created (`/wt 2` instead).
- Teleporting extremely far distances is restricted.
- Some potions are restricted depending on permissions.
- To prevent PvP abuse, `/sit`, `/lay`, `/ded` cannot be used in Survival/Adventure mode.
- `/time` and `/weather` only affect the player who executes them.
- Some items are restricted based on permissions.

## Server Restart

The server runs 24/7.  
However, for stability, it **automatically restarts every day at 3:00 AM**.  

Restart takes about 1 minute, during which login is unavailable.

Sometimes the server may temporarily stop for plugin updates.  
This is intentional and will restart shortly.

- Exact voting time

## Permission Groups

The server has 5 main ranks:

- **Default**  
  Given when joining. Basic commands + build access.

- **Verified**  
  Given after linking Discord. Allows WorldEdit and city creation.

- **Regular**  
  Promoted by staff. Includes OP permissions and command blocks.

- **Moderator**  
  Staff role handling support, griefing, and issues.

- **Admin**  
  Full server management.

Use `/player` to check your rank.

More info:
- [Staff Info](/server/profiles)  
- [Permission Policy](/server/policies/permissions)
